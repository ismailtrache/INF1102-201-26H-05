\# TP Infrastructure as Code – OpenTofu



Ce travail a pour objectif de comprendre le principe de l’Infrastructure as Code (IaC)

en utilisant OpenTofu avec Proxmox VE.



L’IaC permet de créer et gérer des ressources d’infrastructure

de manière automatisée à l’aide de fichiers de configuration.



Ce projet contient :

\- provider.tf : configuration du fournisseur Proxmox

\- main.tf : définition d’une machine virtuelle Linux

\- variables.tf : déclaration des variables

\- terraform.tfvars : valeurs sensibles (non envoyées sur GitHub)
